# Restaurant Game Demo

開創自己的料理王國！在這款模擬遊戲中，您將從一間小餐廳起步，設計裝潢，創造獨特菜單，管理團隊，並逐步擴展至全球。從研發新菜餚到提供卓越的顧客服務，每一個決策都至關重要。接受挑戰，實現餐館夢想，留下在美食世界的印記！
